# Managing Services

<h2><u>Managing Services:</u></h2><p>
	 This is the section you will be able to manage the services you have created (you would create the various services through the quotes and products section).</p><p>
	 <img src="https://s3.amazonaws.com/helpscout.net/docs/assets/5b4c4e430428631d7a88ed10/images/6036c3090a2dae5b58fb3366/file-6WwcIL6lvW.png"></p><p>
	 <span class="wysiwyg-font-size-x-large">Overview</span>&nbsp;You will be able to view&nbsp;the following information in this section:&nbsp;</p><ul>
	
<li>Service ID</li>	
<li>Customer</li>	
<li>Product details</li>	
<li>Type of service</li>	
<li>Description</li>	
<li>Base price</li>	
<li>Final price</li>	
<li>Date created</li>	
<li>Billing cycle&nbsp;</li>	
<li>Service status</li>	
<li>Action icons to edit the services.</li></ul><p>
	 &nbsp;If you click down on the "EDIT" tab on the top right or use the icons on the right of the service,&nbsp;&nbsp;you have the following options:</p><ul>
	
<li>Enable the service</li>	
<li>Disable the service</li>	
<li>Delete the service&nbsp;</li>	
<li>Edit the service&nbsp;</li>	
<li>Unsuspend the service&nbsp;</li>	
<li>Suspend the service</li></ul><p>
	 The filter section next to this, allows you to filter through all the results of the services.</p><p>
	 <img src="https://s3.amazonaws.com/helpscout.net/docs/assets/5b4c4e430428631d7a88ed10/images/6036c4910a2dae5b58fb3376/file-WJD9MaFpoU.png" style="width: 301px;">If you select multiple services to edit you will see the following screen:</p><p>
	 <img src="https://s3.amazonaws.com/helpscout.net/docs/assets/5b4c4e430428631d7a88ed10/images/6036c599661b720174a70a87/file-mW1MaksiGp.png" style="width: 1031px;"></p><p>
	 You are able to action the following in this section:</p><ul>
	
<li>Disable</li>	
<li>Cancel</li>	
<li>Migrate product</li>	
<li>Delete</li></ul><p>
	 If you click next to the action box on the right, you are able to view/edit the full product.</p><p>
	 <img src="https://s3.amazonaws.com/helpscout.net/docs/assets/5b4c4e430428631d7a88ed10/images/6036bfa124d2d21e45ed9e7c/file-Q3VmnQVXkR.png" style="background-color: initial;"></p><p>
	 You will then be directed to the service, where you are able to view and edit the service.</p><p>
	 The following service information will be available:</p><ul>
	
<li>Service Info:</li>	
<li>Customer</li>	
<li>Product</li>	
<li>Status</li>	
<li>Creation date</li>	
<li>Last modified</li>	
<li>Service Type</li></ul><p>
	 <img src="https://s3.amazonaws.com/helpscout.net/docs/assets/5b4c4e430428631d7a88ed10/images/6036c0be661b720174a70a63/file-pdaCtUlHqE.png" style="width: 414px;"></p><p>
	 The following billing information will be available to be edited:</p><ul>
	
<li>Service price</li>	
<li>Discount</li>	
<li>Final price</li>	
<li>Billing cycle</li>	
<li>Billing date&nbsp;</li>	
<li>Billing status</li>	
<li>Last invoice</li>	
<li>Next invoice</li></ul><p>
	 <img src="https://s3.amazonaws.com/helpscout.net/docs/assets/5b4c4e430428631d7a88ed10/images/6036c0e58502d1120e90af4a/file-LChztey9IV.png" style="width: 428px;"></p><p>
	 The next section allows you to see the following information:</p><ul>
	
<li>General info (Radius username/password, Static IP, Active Sessions)</li>	
<li>Radius AVP Pairs - Attributes can be added. Radius attributes specified here will be sent whenever this service connects</li>	
<li>Cap information</li>	
<li>Usage&nbsp;(Can be set to 24 Hours, Last week, or last month)</li>	
<li>Graphs (Can be set to 24 Hours, Last week, or last month)</li>	
<li>Auth logs</li>	
<li>Sessions</li>	
<li>Port locking</li>	
<li>Location</li></ul><p>
	 <img src="https://s3.amazonaws.com/helpscout.net/docs/assets/5b4c4e430428631d7a88ed10/images/6036c1f624d2d21e45ed9e8e/file-tfMcoE5k9y.png">Below this, you are able to add tags, service notes, and add equipment assigned to the service.<img src="https://s3.amazonaws.com/helpscout.net/docs/assets/5b4c4e430428631d7a88ed10/images/6036c2268502d1120e90af5a/file-x86jOtebkX.png"></p><p class="wysiwyg-text-align-center">
	 <em>Please click on our "support" button at the top right of the page if you have not found any information to assist with what you needed in this section.</em></p>
