# Manage Transactions

This section allows you to manage all payment transactions in your iBill.io system, ensuring accurate billing and financial records.

---

## Transactions List Overview
- **Transaction ID**: Unique identifier for each transaction
- **Payment Plugin (Gateway)**: The payment gateway used
- **Customer**: The customer who made the payment
- **Date Created**
- **Amount**
- **Status**
- **Action**: View transaction details

   ![Transactions List](https://s3.amazonaws.com/helpscout.net/docs/assets/5b4c4e430428631d7a88ed10/images/6037e53324d2d21e45eda336/file-9qOfpISoeE.png)

## Viewing Transaction Details
Click on a transaction to view:
- **Payment Plugin (Gateway)**
- **Customer ID**
- **Date Created**
- **Amount**
- **Status**

   ![Transaction Info](https://s3.amazonaws.com/helpscout.net/docs/assets/5b4c4e430428631d7a88ed10/images/6037e60fb3ebfb109b5861cb/file-ddrmUY802f.png)

## Invoices Linked to Transactions
- Click the arrows next to invoice information to view the related invoice.
- From the invoice, you can:
  - Credit items
  - Reverse payments, email, or download invoice payments
  - Add credits
  - Add notes

   ![Linked Invoices](https://s3.amazonaws.com/helpscout.net/docs/assets/5b4c4e430428631d7a88ed10/images/6037e729b3ebfb109b5861d3/file-8Su350ttUd.png)

## Best Practices
- Reconcile transactions regularly
- Use references and notes for clear record-keeping
- Review reversed transactions for potential issues
