# Managing IP Pools

IP Pools are groups of IP addresses that can be dynamically assigned to users and services. Managing IP Pools in iBill.io ensures efficient use of your address space and helps automate service provisioning.

---

## How to Manage IP Pools
- **View IP Pools**: See all pools with details such as ID, pool name, status, and whether the pool is static.
- **Actions**: Use icons to delete or free IPs from pools.

   ![IP Pools](https://s3.amazonaws.com/helpscout.net/docs/assets/5b4c4e430428631d7a88ed10/images/5b4c4fcd0428631d7a88eda9/img-29693-1531727785-666394700.png)

## Creating a New IP Pool
1. **Click + New IP/POOL**
2. **Enter Details**:
   - Pool name (new or existing)
   - First IP address in the pool
   - Last IP address in the pool
3. **Save the Pool**: Click **Add IP** to finalize.

   ![Add IP Pool](https://s3.amazonaws.com/helpscout.net/docs/assets/5b4c4e430428631d7a88ed10/images/5b4c4fcd0428631d7a88edaa/img-29693-1531727785-1147392921.png)

## Best Practices
- Use descriptive names for pools.
- Extend pools by reusing names.
- Regularly review and free unused IPs.
