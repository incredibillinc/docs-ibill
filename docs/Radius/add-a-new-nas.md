# Add a New NAS

A NAS (Network Access Server) is a device that provides access to your network for users and services. Adding NAS devices in iBill.io allows you to manage authentication, accounting, and access control for your ISP network.

---

## How to Add a New NAS

1. **Go to the NAS Management Section**
   - Navigate to Radius > Manage NAS Devices and click **+ New NAS**.

2. **Enter NAS Details**
   - **NAS Name**: A descriptive name for the device.
   - **NAS IP Address**: The IP address of the NAS.
   - **Description**: (Optional) Add notes about the device.

3. **Advanced Options** (optional)
   - Configure POD/COA secrets, targets, NAS type, and walled garden products as needed.
   - Add RADIUS AVP pairs for custom attributes.
   - Set the physical location of the NAS device.

4. **Save the NAS**
   - Click **Create NAS** to add the device to your system.

## Best Practices
- Use clear names and descriptions for each NAS.
- Regularly review and update NAS device settings.
- Use advanced options for complex network setups.
