# Viewing Active Sessions

The Active Sessions page allows you to monitor all currently connected users and services in your network. This is essential for real-time troubleshooting and network management.

---

## What You Can See
- **Session ID**
- **Username**
- **CLI**
- **IP Address**
- **NAS IP**
- **Input MB / Output MB**
- **Start Time / Last Update**
- **Actions (Disconnect)**

   ![Active Sessions](https://s3.amazonaws.com/helpscout.net/docs/assets/5b4c4e430428631d7a88ed10/images/5b4c4fcd2c7d3a03f89c97d5/img-29693-1531727786-956524964.png)

## How to Use
- Monitor session details for all active users.
- Use the Disconnect action to terminate problematic or unauthorized sessions.

## Best Practices
- Regularly monitor active sessions for unusual activity.
- Use session data to troubleshoot connectivity issues.
