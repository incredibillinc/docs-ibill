# Viewing the Session History

The Session History page allows you to review all past user sessions in your network. This is essential for auditing, troubleshooting, and understanding user activity over time.

---

## What You Can See
- **Session ID**
- **Username**
- **CLI**
- **IP Address**
- **NAS IP**
- **Input MB / Output MB**
- **Start Time / Stop Time**

   ![Session History](https://s3.amazonaws.com/helpscout.net/docs/assets/5b4c4e430428631d7a88ed10/images/5b4c4fcc0428631d7a88eda7/img-29693-1531727786-94240369.png)

## How to Use
- Browse or search for specific sessions by user, IP, or time period.
- Use session data to investigate issues, verify usage, or respond to support requests.

## Best Practices
- Regularly review session history for unusual or unauthorized activity.
- Use session logs to support billing, compliance, and customer service.
