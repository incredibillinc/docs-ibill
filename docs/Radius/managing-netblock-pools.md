# Managing Netblock Pools

Netblock Pools are collections of IP address ranges (CIDR blocks) that can be assigned to customers or services. Managing netblocks in iBill.io helps you efficiently allocate and track your IP resources.

---

## How to Manage Netblock Pools
- **View Netblocks**: See all netblocks with details such as CIDR, pool name, assigned entity, type, and status.
- **Actions**: Use icons to delete or free netblocks. Select multiple netblocks for bulk actions.

   ![Netblocks](https://s3.amazonaws.com/helpscout.net/docs/assets/5b4c4e430428631d7a88ed10/images/60413245661b720174a73792/file-Sxe6cIRZsb.png)

## Creating a New Netblock
1. **Click + New Netblock**
2. **Enter Details**:
   - Pool name (new or existing)
   - CIDR notation
   - Netblock type
3. **Save the Netblock**: Click **Add Netblock** to finalize.

   ![Add Netblock](https://s3.amazonaws.com/helpscout.net/docs/assets/5b4c4e430428631d7a88ed10/images/604132d324d2d21e45edcb9d/file-3AvOZdkpzo.png)

## Best Practices
- Use descriptive names for pools and netblocks.
- Regularly review and free unused netblocks.
- Track assignments to avoid conflicts.
