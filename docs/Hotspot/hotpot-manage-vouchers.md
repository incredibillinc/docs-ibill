# Hotspot Manage Vouchers

Hotspot vouchers allow you to provide temporary or prepaid internet access to users. This section explains how to create, manage, and monitor voucher batches in iBill.io.

---

## Hotspot Voucher Batches
- **Create Voucher Batches**: Generate multiple vouchers at once for distribution.
- **View Current Batches**: See all existing batches, including:
  - ID
  - Name
  - Size (number of vouchers)
  - Number of used vouchers
  - Validity period
  - Time and data limits
  - Topup info
  - Status

   ![Voucher Batches](https://s3.amazonaws.com/helpscout.net/docs/assets/5b4c4e430428631d7a88ed10/images/6037ee6524d2d21e45eda399/file-uS3LlSDIkZ.png)

## Batch Actions
- **Delete, Disable, Enable, View**: Use the action icons to manage each batch.

   ![Batch Actions](https://s3.amazonaws.com/helpscout.net/docs/assets/5b4c4e430428631d7a88ed10/images/6037eefd8502d1120e90b493/file-IOPRR5jszr.png)

## Creating a New Voucher Batch
1. **Enter Name and Count**: Specify the batch name and number of vouchers.
   ![Step 1](https://s3.amazonaws.com/helpscout.net/docs/assets/5b4c4e430428631d7a88ed10/images/6037f00624d2d21e45eda3a9/file-6QZJfdLPjt.png)
2. **Set Validity**: Choose how long vouchers are valid (days, hours, months).
   ![Step 2](https://s3.amazonaws.com/helpscout.net/docs/assets/5b4c4e430428631d7a88ed10/images/6037f04424d2d21e45eda3ac/file-BZ6mwhtTI5.png)
3. **Configure Limits**: Set data and time limits for each voucher.
   ![Step 3](https://s3.amazonaws.com/helpscout.net/docs/assets/5b4c4e430428631d7a88ed10/images/6037f0d724d2d21e45eda3b2/file-JX4aFhzA5g.png)
4. **Set Credentials and Topup Options**: Enter username/password or use auto-generation. Choose if vouchers can be used for topups.
   ![Step 4](https://s3.amazonaws.com/helpscout.net/docs/assets/5b4c4e430428631d7a88ed10/images/6037f147661b720174a70f74/file-4vvCRUPVQT.png)
5. **Add RADIUS AVP Pairs** (optional): For advanced configuration.
   ![Step 5](https://s3.amazonaws.com/helpscout.net/docs/assets/5b4c4e430428631d7a88ed10/images/6037f231661b720174a70f79/file-jdjDHc4uLF.png)
6. **Confirm and Generate**: Review details and generate the vouchers. The new batch will appear in the Manage Vouchers section.
   ![Step 6](https://s3.amazonaws.com/helpscout.net/docs/assets/5b4c4e430428631d7a88ed10/images/6037f2a08502d1120e90b4b5/file-vTHSOzmyC2.png)
