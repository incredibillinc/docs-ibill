# Creating a Data Service Product

Data Service products are core offerings for ISPs, allowing you to define recurring or once-off internet access services for your customers. This guide walks you through the steps to create a new Data Service product in iBill.io.

---

## Step-by-Step: Creating a Data Service

1. **Choose Billing Type**
   - Select whether the service is recurring (monthly, yearly, etc.) or once-off.
   - For recurring, set the billing period and start date.

   ![Billing Type](https://s3.amazonaws.com/helpscout.net/docs/assets/5b4c4e430428631d7a88ed10/images/607870d0e0324b5fdfd0b61a/file-IuudgIoJCr.png)

2. **Enter Product Information**
   - Name and describe the product for easy identification.

   ![Product Info](https://s3.amazonaws.com/helpscout.net/docs/assets/5b4c4e430428631d7a88ed10/images/5b4c4fc92c7d3a03f89c97ce/img-2564-1531727767-1661453959.png)

3. **Set Pricing**
   - Add a setup fee (if applicable) and the recurring monthly price (excluding tax).

   ![Pricing](https://s3.amazonaws.com/helpscout.net/docs/assets/5b4c4e430428631d7a88ed10/images/5b4c4fc92c7d3a03f89c97cf/img-2564-1531727767-698777546.png)

4. **Configure Service Details**
   - Optional: Add service realm details, session limits, and select IP information.

   ![Service Details](https://s3.amazonaws.com/helpscout.net/docs/assets/5b4c4e430428631d7a88ed10/images/5b4c4fc90428631d7a88eda2/img-2564-1531727767-831496074.png)

5. **Select Service Type**
   - Choose from capped or uncapped (basic/advanced) service types.

   ![Service Type](https://s3.amazonaws.com/helpscout.net/docs/assets/5b4c4e430428631d7a88ed10/images/5b4c4fc92c7d3a03f89c97d0/img-2564-1531727767-1857650730.png)

6. **Set Data Allocations (if capped)**
   - Define cap size, refresh period, and rollover options.

   ![Data Allocations](https://s3.amazonaws.com/helpscout.net/docs/assets/5b4c4fca0428631d7a88eda3/img-2564-1531727767-17354478.png)

7. **Configure RADIUS Attributes**
   - Set upload/download speed and other advanced options (e.g., for Mikrotik NAS).

   ![RADIUS Attributes](https://s3.amazonaws.com/helpscout.net/docs/assets/5b4c4fca2c7d3a03f89c97d1/img-2564-1531727767-1944871208.png)

8. **Review and Confirm**
   - Check all details and click Create Product to finalize.

   ![Confirm Product](https://s3.amazonaws.com/helpscout.net/docs/assets/5b4c4fca2c7d3a03f89c97d2/img-2564-1531727767-158024310.png)
