# Creating a Static IP Product

Static IP Products allow you to assign fixed IP addresses to customers for advanced networking needs. This guide explains how to create a Static IP Product in iBill.io.

---

## Step-by-Step: Creating a Static IP Product

1. **Set Billing Information**
   - Choose recurring or once-off billing, set the period and billing date.

   ![Billing Info](https://s3.amazonaws.com/helpscout.net/docs/assets/5b4c4e430428631d7a88ed10/images/5b4c4fc12c7d3a03f89c97bb/img-2564-1531727770-295723608.png)

2. **Enter Product Information**
   - Name and describe the product.

   ![Product Info](https://s3.amazonaws.com/helpscout.net/docs/assets/5b4c4e430428631d7a88ed10/images/5b4c4fc22c7d3a03f89c97bc/img-2564-1531727770-1611646109.png)

3. **Set Pricing**
   - Add setup fee (if any) and monthly price.

   ![Pricing](https://s3.amazonaws.com/helpscout.net/docs/assets/5b4c4e430428631d7a88ed10/images/5b4c4fc20428631d7a88ed97/img-2564-1531727770-371302079.png)

4. **Select IP Pool**
   - Choose the IP pool from which static IPs will be assigned.

   ![IP Pool](https://s3.amazonaws.com/helpscout.net/docs/assets/5b4c4e430428631d7a88ed10/images/5b4c4fc22c7d3a03f89c97bd/img-2564-1531727770-1690853744.png)

5. **Review and Confirm**
   - Review all details and click Create Product to finalize.

   ![Confirm Product](https://s3.amazonaws.com/helpscout.net/docs/assets/5b4c4fc22c7d3a03f89c97be/img-2564-1531727770-596504288.png)
