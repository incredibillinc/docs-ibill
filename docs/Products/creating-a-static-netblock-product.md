# Creating a Static Netblock Product

Static Netblock Products allow you to allocate a range of static IP addresses (netblocks) to customers for advanced networking scenarios. This guide explains how to create a Static Netblock Product in iBill.io.

---

## Step-by-Step: Creating a Static Netblock Product

1. **Set Billing Information**
   - Choose once-off or recurring billing as needed.

   ![Billing Info](https://s3.amazonaws.com/helpscout.net/docs/assets/5b4c4e430428631d7a88ed10/images/5b4c4fbc2c7d3a03f89c97b0/img-2564-1531727771-1204739091.png)

2. **Enter Product Information**
   - Name and describe the product.

   ![Product Info](https://s3.amazonaws.com/helpscout.net/docs/assets/5b4c4e430428631d7a88ed10/images/5b4c4fbd2c7d3a03f89c97b1/img-2564-1531727771-1486539523.png)

3. **Set Pricing**
   - Enter the one-time or recurring price.

   ![Pricing](https://s3.amazonaws.com/helpscout.net/docs/assets/5b4c4e430428631d7a88ed10/images/5b4c4fbd0428631d7a88ed8d/img-2564-1531727771-2046319748.png)

4. **Select Netblock Pool**
   - Choose the netblock pool for allocation.

   ![Netblock Pool](https://s3.amazonaws.com/helpscout.net/docs/assets/5b4c4e430428631d7a88ed10/images/5b4c4fbd2c7d3a03f89c97b2/img-2564-1531727771-660680463.png)

5. **Review and Confirm**
   - Review all details and click Create Product to finalize.

   ![Confirm Product](https://s3.amazonaws.com/helpscout.net/docs/assets/5b4c4fbd0428631d7a88ed8e/img-2564-1531727771-932744079.png)
