# Creating a Data Topup Product

Data Topup products allow your customers to purchase additional data when they reach their cap. This guide explains how to create a new Data Topup product in iBill.io.

---

## Step-by-Step: Creating a Data Topup Product

1. **Enter Product Information**
   - Name and describe the topup product.

   ![Product Info](https://s3.amazonaws.com/helpscout.net/docs/assets/5b4c4e430428631d7a88ed10/images/5b4c4fc30428631d7a88ed98/img-2564-1531727769-1692661642.png)

2. **Set Pricing**
   - Enter the once-off price for the topup.

   ![Pricing](https://s3.amazonaws.com/helpscout.net/docs/assets/5b4c4e430428631d7a88ed10/images/5b4c4fc32c7d3a03f89c97c2/img-2564-1531727769-1546423540.png)

3. **Configure Topup Settings**
   - Set the data amount, validity period (e.g., 3 months from purchase), and whether it expires on next recharge or after a specific period.

   ![Topup Settings](https://s3.amazonaws.com/helpscout.net/docs/assets/5b4c4e430428631d7a88ed10/images/5b4c4fc30428631d7a88ed99/img-2564-1531727769-816370478.png)

4. **Finalize and Save**
   - Review all details and create the product. You will be directed to the Manage Products page.
